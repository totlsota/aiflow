# AI demand chain

Static single-page webapp for Vercel.

## Deploy

Upload this folder to Vercel as a static project. No framework preset and no build command are required.

## Contents

- `index.html` — the complete app, including the embedded Sankey data and source links.
- `vercel.json` — simple SPA fallback rewrite.

Most claim chips link to the original source article. PDF-derived summaries fall back to their ClarityReader page.
